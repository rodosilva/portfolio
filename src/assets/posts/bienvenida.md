# Bienvenido a mi Blog

Este es mi primer artículo en este blog. He creado este sitio para compartir mis ideas y experiencias sobre desarrollo web.

## ¿Cómo funciona?

Este blog está construido con:

- **React** para la interfaz de usuario
- **Vite** para un desarrollo rápido
- **TailwindCSS** para los estilos
- **Markdown** para los contenidos

## Características

1. **Lectura de archivos Markdown** - Los posts se cargan como archivos markdown
2. **Categorías con Tags** - Puedes filtrar posts por etiquetas
3. **Diseño Responsivo** - Funciona en dispositivos móviles y de escritorio
4. **Dark Mode** - Soporte para tema oscuro
